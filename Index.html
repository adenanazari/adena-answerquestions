Part 2:

Front-end developer: A front-end developer is a type of web developer that focuses on creating components of a website or application for the user. Front-end developers are using languages such as HTML, CSS, and JavaScript.

QA Engineer:

QA Engineer creates tests that identify issues with software before the products are released. They plan ideas to improve the quality of the product and resolve bugs.

Cloud Engineer:

A cloud engineer is an IT professional who is responsible for any technical tasks related to cloud computing, including design, planning, management, maintenance, and support. They work with cloud platforms such as Amazon Web Service (AWS), Microsoft Azure, and Google Cloud Platform to make sure that all applications run smoothly and correctly in the cloud.

Product Manager:

A product manager is responsible for the management and development of software from the start to the final release. They work with designers, developers, and other stakeholders to define the product vision.

UX/UI Designer:

The UX designer is responsible for making a product or service usable, enjoyable, and accessible. They work closely with developers, product managers, and other stakeholders to make sure that the design meets the needs of the users.

Back End Developer:

Back-end developers use languages like Java, Python, or Ruby. They are responsible for designing, implementing, and maintaining the server-side logic and database of a website or application.
